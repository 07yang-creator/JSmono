# JSmono — マイソク自動生成システム

## プロジェクト概要

日本の不動産業界で広く使われている「マイソク（物件概要書）」を、エージェントが簡単に作成・保存・PDF出力できるWebアプリケーション。

エージェントは物件情報をフォームに入力し、写真・地図画像をアップロードするだけで、印刷可能なA4サイズのマイソクPDFが生成される。

---

## 開発背景

- マイソクはすべての不動産エージェントが毎物件作成する標準的な1枚書類
- 現在は手作業またはDesktopソフトで作成しており、非効率
- レイアウト・構成はほぼ業界標準で固定されており、自動化に適している
- 本システムはそのプロセスをWebアプリとして標準化・効率化する

---

## ターゲットユーザー

- 不動産エージェント（個人・法人問わず）
- アカウント登録制（メール＋パスワード）
- 各エージェントが自社ブランディング（ロゴ・会社情報）を登録して使用

---

## 開発フェーズ

### Phase 1 — コア機能（現在の開発対象）
- ユーザー認証（登録・ログイン）
- エージェントプロフィール設定
- 物件情報入力フォーム（全フィールド・物件種別による条件表示）
- テンプレート1のプレビュー表示
- PDF出力（A4印刷対応）
- 過去物件の保存・編集・複製・削除

### Phase 2 — テンプレート拡張
- テンプレート2：写真重視レイアウト
- テンプレート3：情報量重視レイアウト
- テンプレート4：マンション専用レイアウト
- 右カラムブロックのドラッグ並び替え

### Phase 3 — 有料ユーザー向け高度機能
- カスタムテンプレートのアップロード
- フォント・カラー・強調の詳細カスタマイズ
- チーム共有・複数エージェント管理

---

## テクノロジースタック

| レイヤー | 使用技術 |
|---|---|
| フロントエンド | React + Tailwind CSS |
| 認証・データベース | Supabase (PostgreSQL) |
| ファイルストレージ | Supabase Storage |
| PDF出力 | html2pdf.js |
| ホスティング | Vercel |
| 言語 | 日本語のみ |
