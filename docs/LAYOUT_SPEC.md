# レイアウト仕様書 — テンプレート1

A4縦（210mm × 297mm）、印刷対応。
テンプレート1はサンプルマイソク（新宿区北新宿3丁目）をベースとした標準レイアウト。

## 各ゾーン詳細

### ゾーン1: 左上 — 物件ヘッダー
- 背景: ブランドカラー（エージェント設定）
- 物件種別バッジ（例: 中古戸建）: 白文字、小サイズ
- 物件タイトル（A2）: 白文字、大サイズ、太字 — **常に強調表示**
- 交通アクセス（B1〜B3）: 白文字、小サイズ
- 「販売価格」ラベル: ブランドカラーの帯
- 価格（A4）: 大サイズ、太字 — **常に強調表示**

### ゾーン2: 左中〜左下 — 物件概要
- 「所在地」セクションラベル: ブランドカラー帯
- 住所（A3）: 標準テキスト
- 「物件概要」セクションラベル: ブランドカラー帯
- 【土地】【建物】等サブラベル: 太字
- 各フィールド: ■ラベル + 値のペア形式
- 空フィールドは非表示（行ごと消える）

### ゾーン3: 中央上 — 外観写真（K1）
- object-fit: cover でスロットを埋める
- 未アップロード時: グレーのプレースホルダー表示

### ゾーン4: 右上 — オプションボックス
- G（借地条件）が有効な場合: 借地期間・地代を表示
- G無効の場合: K3（写真3）またはK4（間取り図）を表示
- 何もない場合: 非表示、スペースをK1が吸収

### ゾーン5: 中央中 — 周辺環境・学区（H/I）
- 「周辺環境」タイトル: 大きめ、アイコン付き
- H1〜H8: 施設名 + 距離のリスト
- 「学区」タイトル
- I1〜I2: 学校名 + 距離
- I非表示の場合: Hがスペースを吸収

### ゾーン6: 右中 — 地図（K5）
- 地図画像をフルスロット表示
- object-fit: cover
- 「地図」ラベル付き

### ゾーン7: 右下 — 写真2（K2）
- K2未アップロード時: 非表示
- K5とK2のスペース比は 60:40

### ゾーン8: フッター（全幅）
- 背景: ブランドカラー（薄め）または白
- 左: 会社ロゴ（L3）+ 会社名（L1）+ 住所（L4）
- 中央: 「お問い合わせ先」ラベル + 電話（L5）+ FAX（L6）
- 右: 担当者名（L8）+ 直通（L9）+ メール（L7）
- 最下部細帯: 免許番号（L10）+ 所属団体（L11）+ 取引態様

## オートフィット仕様

### テキスト縮小ロジック
左カラムの物件概要エリアがオーバーフローした場合:
1. フォントサイズを 100% → 95% → 90% → 85% の順に縮小
2. 各ステップでオーバーフロー再チェック
3. 85%でもオーバーフローの場合: 警告表示 + グローバルフォントスライダーを表示

### 画像フィット
- 全画像スロット: `object-fit: cover` で比率を保ちながらスロットを埋める
- 縦横比の強制なし、中央クロップ

### 空フィールドの処理
- optional フィールドが空の場合: その行を非表示（`display: none`）
- セクション全体が空の場合: セクションラベルごと非表示

## カラーテーマ

エージェントがプロフィールで設定する1色（HEXコード入力）が以下に適用:
- セクションラベル帯の背景色
- 物件タイトルバナーの背景色
- フッターの背景色（透明度50%）
- アクセントライン・ボーダー

その他はすべて黒テキスト・白背景。

## フォント設定

### 選択肢（3種）
| 選択 | タイトル系 | 本文系 | 印象 |
|---|---|---|---|
| Gothic（デフォルト） | Noto Sans JP | Noto Sans JP | モダン・クリーン |
| Mincho | Noto Serif JP | Noto Serif JP | 伝統・フォーマル |
| Mixed | Noto Serif JP | Noto Sans JP | 高級感・読みやすさ両立 |

### サイズ階層
| 用途 | サイズ | スタイル |
|---|---|---|
| 物件タイトル（A2） | 22px | 太字、常に強調 |
| 販売価格（A4） | 28px | 太字、常に強調 |
| セクションラベル | 11px | 太字、白抜き |
| サブラベル（【土地】等） | 10px | 太字 |
| 本文フィールド | 9px | 標準 |
| フッター | 8px | 標準 |

## 強調表示オプション（Phase 3実装予定）

エージェントが任意フィールドに適用可能:
| 方法 | 効果 |
|---|---|
| カラー | テキストをブランドカラーに変更 |
| 拡大 | フォントサイズ +2ステップ |
| アンダーライン | ブランドカラーの下線 |
| ボックス | 薄いカラー背景ボックス |
| 太字 | フォントウェイト増加 |
