## テンプレート1 レイアウト仕様

### グリッド構造（A4縦 210×297mm）

```
┌──────────────────────┬──────────────┬───────────────┐  ↑
│ A1: 物件種別バッジ     │              │               │  |
│ A2: 物件名（大）       │  K1: 外観写真 │  G: 借地条件   │  ~35%
│ B: 交通アクセス        │   (メイン)    │  または空白    │  |
│ A4: 販売価格（特大）   │              │               │  ↓
├──────────────────────┼──────┬───────┴───────────────┤  ↑
│ A3: 所在地ラベル       │      │                       │  |
├──────────────────────┤  H   │   K5: 地図画像         │  |
│ 【土地】 C            │  周辺 │                       │  |
│ 【建物】 D            │  環境  ├───────────────────────┤  ~55%
│ 【マンション】E        │      │                       │  |
│ 【法令制限】F          │  I   │   K2: 外観写真2        │  |
│ 【備考】J             │  学区 │   (optional)          │  |
│                      │  J   │                       │  ↓
├──────────────────────┴──────┴───────────────────────┤  ↑
│ L: 会社情報フッター（全幅）                             │  ~10%
└─────────────────────────────────────────────────────┘  ↓

左カラム幅: ~38%
中カラム幅: ~12%（H/I/J）
右カラム幅: ~50%（写真・地図）
```

### 自動フィット仕様（Phase 1）

1. 各セクションはデフォルトフォントサイズ（100%）でレンダリング
2. 左カラムがオーバーフローした場合、段階的に縮小:
   - Step 1: 95%
   - Step 2: 90%
   - Step 3: 85%
3. 85%でもオーバーフローの場合、オレンジ色の警告を表示:
   「内容が多すぎます。フォントサイズを調整するか、備考を短縮してください。」
4. エージェントがグローバルフォントスライダーで手動調整可能（70%〜100%）
5. 画像スロット: `object-fit: cover` でトリミング表示
6. 空のoptionalセクション: 非表示、スペース吸収

## デザイン仕様

### カラーテーマ
- エージェントがブランドカラーを1色選択（HEXコード直接入力）
- 適用箇所:
  - セクションヘッダーバー背景
  - A1物件種別バッジ背景
  - A4価格テキスト色
  - フッターL背景
  - アクセントライン・ボーダー
- その他: 黒テキスト `#1a1a1a`、白背景 `#ffffff`

### フォントシステム
| 選択肢 | タイトル | 本文 |
|---|---|---|
| Gothic（デフォルト） | Noto Sans JP Bold | Noto Sans JP Regular |
| Mincho | Noto Serif JP Bold | Noto Serif JP Regular |
| Mixed | Noto Serif JP Bold | Noto Sans JP Regular |

### 固定強調（変更不可）
- A2 物件名: 常に `font-size: 2rem`, `font-weight: bold`
- A4 販売価格: 常に `font-size: 2.5rem`, `color: ブランドカラー`, `font-weight: bold`

### 任意強調オプション
エージェントが任意のフィールドを選択して適用可能:

| 強調タイプ | CSS適用 |
|---|---|
| カラー変更 | `color: ブランドカラー` |
| 文字拡大 | `font-size: +0.2rem` |
| 下線 | `text-decoration: underline` |
| ボックスハイライト | `background: ブランドカラー20%透明度` |
| 太字 | `font-weight: bold` |

## PDF出力仕様

- ライブラリ: `html2pdf.js`
- 用紙サイズ: A4（210×297mm）
- 向き: 縦
- 解像度: 150dpi（印刷品質）
- マージン: 8mm（上下左右）
- 出力ファイル名: `{物件名}_{日付}.pdf`
- フォント埋め込み: Google Fonts経由でNoto JP系を使用
